This repository contains the supplementary data for the referenced paper below. The calibrated models presented in the paper are
implemented in the BED-SimDesign framework as one of its concentrated-plasticity modelling options.


Journal Paper

Hasanoglu, S. and O'Reilly, G. J. (2026). A hysteretic energy–based framework for seismic fragility assessment of ductile reinforced
concrete frame buildings. (UNDER REVIEW)


How to use

To apply the calibrated models in a structural analysis workflow, use the BED-SimDesign framework with the cp02 modelling option.

BED-SimDesign: https://simdesign.builtenvdata.eu
